# Descrição do Problema

### Problema de negócio
> **CSD:** Certeza
> **Fonte:** Sabatina
> **Iteração:** 0

A taxa de conversão do onboarding é de 12% (meta: 40%). Usuários abandonam após o cadastro sem completar a primeira tarefa. Isso representa ~R$ 45k/mês em receita não capturada.

### Problema do usuário
> **CSD:** Suposição
> **Fonte:** Setup
> **Iteração:** 0

Usuários não entendem o que o produto faz ao entrar pela primeira vez. A interface assume conhecimento prévio do fluxo logístico que novos usuários não têm. Baseado em feedback anedótico do suporte — sem dados de pesquisa.
